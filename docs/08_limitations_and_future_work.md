# 8. Limites et Améliorations Futures

## 8.1 Limites

- Dépendance au LLM
- Possible imprécision sémantique
- Pas d’analyse causale automatique

## 8.2 Améliorations futures

- Ajout de tests unitaires
- Détection automatique du type de question
- Interface React avancée
- Caching des réponses LLM
- Analyse statistique automatique

# 7. Moteur de Visualisation

Les visualisations sont générées via Plotly Express.

## 7.1 Types supportés

- Bar chart
- Line chart
- Scatter plot
- Boxplot
- Histogram
- Heatmap

## 7.2 Agrégation

Les agrégations possibles :
- mean
- sum
- median
- count

Cela permet de répondre à des problématiques de comparaison ou d’évolution.

## 7.3 Bonnes pratiques intégrées

- Tri automatique
- Limitation top_k
- Labels personnalisés
- Marges propres
